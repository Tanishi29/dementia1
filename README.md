# p5.js

This is the Template Repl for p5.js.

p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.js is free and open-source because we believe software, and the tools to learn it, should be accessible to everyone. This platform was used to make an application for dementia patients to facilitate better thinking, memory improvements while they enjoy and learn from the game. 

[Check out the official docs here](https://p5js.org/reference/).
